HTML native elements

Regular HTML native elements are already customized and can be written as normal HTML pages (the elements in this list can be used directly as element names).

Abbr.

abbr

Bold

b, bold and strong

Italic

i and italic

Strikethrough

st, s, lt and strike

Overline

overline and ol

Underline

underline and ul
Note: ul is not supported as an element name because it conflicts with HTML native element.
